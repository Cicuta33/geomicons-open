Geomicons Open

Open Source Icons for the Web v0.0.1

Fast

Smaller than similar webfonts and fewer network requests than other icon solutions, Geomicons Open are built from the ground up for performance. Designed on a grid for pixel-precision at standard sizes, each icon is hand-coded to avoid any bloat created from traditional vector drawing applications.
